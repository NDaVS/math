\chapter{Заключение}
Программа иллюстрирует взаимодействие между задачами, выполняемыми асинхронно, и системой планирования. Она демонстрирует:
\begin{itemize}
	\item использование функциональных интерфейсов для задания логики поиска;
	\item контроль выполнения через объект \texttt{Future};
	\item безопасное завершение и отмену задач по таймеру.
\end{itemize}
