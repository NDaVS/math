\chapter{Заключение}
Программа демонстрирует:
\begin{itemize}
	\item создание и использование \texttt{CompletableFuture};
	\item реактивную обработку результата без явного ожидания завершения потока;
	\item использование функционального интерфейса \texttt{Supplier<T>} для передачи логики в асинхронный поток.
\end{itemize}
Таким образом, данный пример иллюстрирует современный подход к организации асинхронных вычислений в Java без необходимости вручную управлять потоками.
